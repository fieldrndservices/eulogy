Dig VI

Owning Palette: Eulogy VIs

Requires: Eulogy

A VI script that encloses a selection of code in a Diagram Disable Contional structure, creates a duplicate of the original code to be refactored for the Enabled frame, adds an "Original" and "Refactored" subdiagram label to each frame, respectively, and adds a bookmark with "RIP", the DoD (Date of Death), and grave digger (programmer) as the structure's label.

By default, today's date is used as the DoD and the application's user name is used for the grave digger.

VI InA reference to the VI where a grave will be dug.
Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Selection List[] References to the GObjects in the VI's block diagram that should be placed in a grave.
Error Out contains error information. This output provides standard error out functionality.
VI OutA duplicate reference to the VI where the grave was dug.
