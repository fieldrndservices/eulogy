Is Grave VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Checks if a GObject is a grave. A grave is Diagram Disable Structure with a RIP bookmark tag for its label, a.k.a. tombstone.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
GObject In is a reference to any object that could be a grave.
Grave? is TRUE if the reference to an object is a grave; otherwise, it is FALSE.
Error Out contains error information. This output provides standard error out functionality.
GObject Out is a duplicate of the reference to any object that could be a grave.
