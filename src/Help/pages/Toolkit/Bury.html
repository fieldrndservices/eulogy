Bury VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Burys the corpse of legacy code. This removes the grave (Diagram Disable Structure) and leaves behind the Spirit frame, which should be enabled before burying the corpse. Note, the corpse frame will no longer exists and be pernamently deleted.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Graves contains references to Diagram Disable Structures with a RIP bookmark tag for their labels.
Error Out contains error information. This output provides standard error out functionality.
Change Made? indicates if any corpses were buried at any of the graves. This will return TRUE if any Diagram Disable Structure was removed from the block diagram.
