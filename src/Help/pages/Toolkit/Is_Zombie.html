Is Zombie VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Checks if any object is a grave with a zombie. Graves are Diagram Disable Structures with a RIP bookmark tag for their labels (tombstone), and a zombie is a grave with its Corpse (legacy) frame enabled and Spirit (refactored) frame disabled.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
GObject In is a reference to any object that could be a grave with a zombie.
Zombie? is TRUE if the reference to an object is a grave with a zombie; otherwise, it is false.
Error Out contains error information. This output provides standard error out functionality.
GObject Out is a duplicate of the reference to any object that could be a grave with a zombie.
