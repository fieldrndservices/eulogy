Reanimate VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Creates zombies at each grave. Enables the Corpse (legacy) frame and disables the Spirit (refactored) frame of each grave. A grave is Diagram Disable Structure with a RIP bookmark tag for its label, a.k.a. tombstone. The Corpse (legacy) frame's subdiagram label is changed from "Corpse" to a Zombie bookmark tag with an ISO-8601 timestamp for its "rebirthday" and the user that created the zombie, a.k.a. the necromancer, surrounded by parentheses.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Graves contains references to Diagram Disable Structures with a RIP bookmark tag for their labels.
Error Out contains error information. This output provides standard error out functionality.
Change Made? indicates if any corpses were reanimated at any of the graves. This will return TRUE if the Corpse (legacy) frame of any Diagram Disable Structure was enabled and the Spirit (refactored) frame was disabled.
