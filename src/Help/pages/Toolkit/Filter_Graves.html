Filter Graves VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Filters an array of GObject references and keeps only references to graves. Any references that are NOT to a grave are closed. A grave is a Diagram Disable Structure with a RIP bookmark tag for its label, a.k.a. tombstone.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
GObjects contains references to any GObjects that might be graves.
Error Out contains error information. This output provides standard error out functionality.
Graves contains references only to Diagram Disable Structures that were identified as graves.
