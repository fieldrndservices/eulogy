Find Zombies VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Finds all graves with a zombie in the block diagram of a VI. Graves are Diagram Disable Structures with a RIP bookmark tag for their labels (tombstone), and a zombie is a grave with its Corpse (legacy) frame enabled and Spirit (refactored) frame disabled.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
VI In is a reference to the VI that is to be searched.
Error Out contains error information. This output provides standard error out functionality.
VI Out is a duplicate of the reference to the VI that was searched.
Graves contains references to Diagram Disable Structures that are graves with an enabled Corpse frame, a.k.a. zombie.
