Find Graves VI

Owning Palette: Eulogy VIs

Requires: Eulogy

Finds all graves in the block diagram of a VI. A grave is a Diagram Disable Structure with a RIP bookmark tag for its label, a.k.a. tombstone.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
VI In is a reference to the VI that is to be searched.
Error Out contains error information. This output provides standard error out functionality.
VI Out is a duplicate of the reference to the VI that was searched.
Graves contains references to Diagram Disable Structures that are graves.
