Bookmark Manager (Graveyard)

Owning Palette: Eulogy Add-on

Requires: Eulogy, Base Development Environment

The Eulogy add-on comes with a custom Bookmark Manager, a.k.a. Graveyard, that makes finding and managing graves easier.

Select View>>Bookmark Manager to display the Select a Bookmark Manager window. Select the Graveyard from from the list and click OK button to display the Graveyard. Use this window to find and display all graves and zombies in your application instance.

  • Scope - Lists projects and VIs in the current application instance.
  • Bookmarks - Lists all graves and zombies in the current application instance. Double-click a grave or zombie to locate it on the block diagram of the VI. Right-click on a grave or zombie to display a shortcut menu to reanimate, smite, or bury a zombie or grave, respectively.
  • Group - Allows you to display the bookmarks by either graves or zombies.

The Graveyward window displays an error message if LabVIEW cannot display certain graves or zombies. Refer to the following list of possible causes for graves or zombies not appearing in this window:

  • VIs are saved in an earlier version of LabVIEW. Load the VIs into memory or mass-compile them into the current LabVIEW version.
  • VIs are password-protected. Add the passwords to the password cache.
  • VIs are locked. Unlock the VIs to view the bookmarks.
