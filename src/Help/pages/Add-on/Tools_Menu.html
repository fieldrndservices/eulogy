Tools Menu

Owning Palette: Eulogy Add-on

Requires: Eulogy, Base Development Environment

Eulogy adds a Field R&D Services>>Eulogy sub-menu to the Tools Menu that includes items for working with legacy code.

  • Create a Hook... - Displays a dialog to create a hook VI for a Eulogy-related action, such as Burying, Digging, and Reanimating.
  • Graveyard... - Displays the custom Bookmark Manager for monitoring and managing graves and zombies in projects and VIs.

Use the pull-down menu to select a Tools Menu action.

Create a Hook...

This Tools menu action shows a dialog for creating a hook VI that is executed either before (pre) or after (post) a Eulogy-related action. Given all of the criteria for creating a hook, this Tools menu action will display a dialog that helps with the creation of a new hook for an Eulogy-related action. The dialog will ensure the VI is properly named based on the selected action, placed in an appropriate location, and generate a template VI with the appropriate controls, indicators, connection pane, etc. The generated template VI will also contain comments and instructions to directly modify the block diagram.

To execute this Tools Menu action:

  1. Start LabVIEW.
  2. Select Tools>>Field R&D Services>>Eulogy>>Create a Hook... from the menu bar. The "Create a Hook" dialog will be displayed.
  3. Select the appropriate option for each input.
  4. Click CREATE to generate a hook VI. The dialog will disappear.
  5. Open the generated hook VI in LabVIEW and modify the contents of its block diagram according to the comments and instructions.
  6. Save the modified hook VI.
  7. Execute a Eulogy-related action.

Graveyard...

This Tools Menu action launches the custom bookmark manager for viewing graves and zombies in LabVIEW projects and VIs. The Graveyard provides addition features and actions to show graves bury corpses, reanimate corpses, and smite zombies using the run-time shortcut menu for tree items. This is the same as using opening the custom bookmark manager via View>>Bookmark Manager menu item and selecting Graveyard from the Select a Bookmark Manager dialog that appears.

To execute this Tools Menu action:

  1. Start LabVIEW.
  2. Select Tools>>Field R&D Services>>Eulogy>>Graveyard... from the menu bar. The Graveyard dialog will be displayed.
